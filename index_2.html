Base URL Structure

https://steveyoung74.github.io/digitalforms/modular_form_joint.html?modules=MODULE_LIST&journey=JOURNEY_NAME&applicationType=individual

Application Types

Parameter Value Description Behavior
individual Single applicant Shows modules once for one person
joint Joint application (e.g., husband & wife) Duplicates customer modules for Customer 1 and Customer 2

Available Modules

Module ID Module Name Description
customer-identification Customer Identification Name, DOB, birth place
contact-details Contact Details Address, phone, email, marital status
previous-address Previous Address Historical address information
tax-residency Tax Residency Tax citizenship and TIN details
us-person US Person Status FATCA compliance questions
marketing Marketing Preferences Communication opt-in/out
declaration Declaration & Signature Legal declaration and e-signature

n8n Integration

JavaScript code for n8n workflow node:

const baseUrl = 'https://steveyoung74.github.io/digitalforms/modular_form.html'; const modules = 'customer-identification,contact-details,declaration'; const journey = 'New Client'; const params = new URLSearchParams({ modules: modules, journey: journey, firstName: $json.firstName, email: $json.email }); return { json: { url: `${baseUrl}?${params}` }};
