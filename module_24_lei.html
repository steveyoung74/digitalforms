What is an LEI?

A Legal Entity Identifier (LEI) is a 20-character, alpha-numeric code that uniquely identifies legally distinct entities that engage in financial transactions.

LEI requirements:

  • Required for entities trading in financial instruments
  • Must be renewed annually to remain valid
  • Format: 20 alphanumeric characters (e.g., 549300ABCDEFGHIJ1234)
