Module 25 - Trust Entity Classification

About FATCA/CRS Classification

The Foreign Account Tax Compliance Act (FATCA) and Common Reporting Standard (CRS) require proper classification of trust entities for international tax compliance.

Please select an option

Participating Financial Institution

Trust is registered with IRS as Participating FI with a valid GIIN

Trustee Documented Trust

Trust with corporate trustee who provides FATCA compliance

Non-Participating Financial Institution

Trust does not participate in FATCA reporting (contact investment manager required)

Owner Documented Financial Institution

Trust qualifies for owner documentation requirements

Non-Reporting Financial Institution

Trust is exempt from FATCA/CRS reporting

Active Non-Financial Foreign Entity (NFFE)

Trust is not a financial institution and is actively managed
